## 1. Introduction
### 1.1 Purpose
This procedure defines the requirements and process for software development lifecycle activities, in accordance with IEC 62304.


### 1.2 Scope
This procedure applies to software development activities for medical device software. This procedure covers software development planning, requirements, architecture, implementation, integration, and testing. Software problem resolution is covered by the Software Problem Resolution Procedure.
