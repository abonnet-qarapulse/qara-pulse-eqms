# Change Control Procedure

## 1. Introduction
### 1.1 Purpose
This procedure defines the process for evaluating, approving, and implementing changes to QMS documents, products, processes, and related systems.


### 1.2 Scope
This procedure applies to all changes affecting controlled documents, product specifications, manufacturing processes, software, and QMS processes. Emergency changes and changes covered by specific procedures (e.g., design changes) are handled according to those procedures.
