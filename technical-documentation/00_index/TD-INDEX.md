# TD Index

Entry point for the Technical Documentation structure aligned with MDR Annex II & III.

| Section | Folder | Purpose |
|---|---|---|
| Introduction | `01_introduction/` | TD overview and scope |
| Device description | `02_device-description/` | Intended purpose, classification, specifications |
| Information supplied | `03_information-supplied/` | Labeling, IFU, user-facing information |
| Design & manufacturing | `04_design-and-manufacturing/` | Design outputs, architecture, development evidence |
| GSPR | `05_gspr/` | GSPR checklist and mapping to evidence |
| Risk management | `06_risk-management/` | Risk management file structure and outputs |
| Verification & validation | `07_verification-validation/` | V&V plans and reports, clinical evaluation |
| Post-market | `08_post-market/` | PMS/PMCF planning and reporting |
| General documentation | `09_general-documentation/` | Declarations, certificates (as applicable) |
| Appendices | `99_appendices/` | Annexes and supporting materials |
